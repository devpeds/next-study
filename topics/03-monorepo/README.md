# [Topic 03] Monorepo

Next.js app generated by `pnpm turbo gen create-topic`

## Getting Started

```bash
pnpm install

pnpm dev # run server in development mode (http://localhost:3000)

# or
pnpm build # build the app
pnpm start # run server in production mode (http://localhost:3000)
```
